C++中vector的常用函数（二）——unique

发表于 2023-02-24 分类于 C++
本文字数： 2.1k 阅读时长 ≈ 2 分钟

unique方法简介

从头到尾遍历容器（该容器必须为有序容器），判断当前元素是否等于上一个元素，将不重复的元素移到前面来(赋值)，而不是将重复的元素移动到后面去。

函数原型

unique(
Iterator first,
Iterator last
);

unique(
Iterator first,
Iterator last,
BinaryPredicate pred
);

first：去重的起点的迭代器位置。

last：去重的终点的迭代器位置，不包括

pred：自定义判断重复方式。

去重区间：【first，last）

返回值：去重以后vector中没有重复元素的下一个位置的迭代器
